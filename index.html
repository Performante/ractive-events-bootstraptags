Ractive.js bootstraptags event plugin

download: ractive-events-bootstraptags.js more plugins at docs.ractivejs.org/latest/plugins

Small events plugin, that adds event functionality of Bootstrap Tags Input plugin

Supports events: beforeItemAdd, itemAdded, beforeItemRemove and itemRemoved

Decorator "tagsinput" comes from decorator plugin
